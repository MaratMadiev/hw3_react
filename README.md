
# React + TypeScript + Vite
## E-commerce Admin Panel

Административная панель для e-commerce системы на React + TypeScript.

## Технологии

- React 18
- TypeScript
- Redux Toolkit + RTK Query
- React Router v6
- Redux Persist
- Feature-Sliced Design (почти)

## Функциональность

- ✅ Авторизация (DummyJSON API)
- ✅ Просмотр списка продуктов с пагинацией
- ✅ Поиск продуктов
- ✅ Детальная страница продукта
- ✅ Профиль пользователя
- ✅ Защищенные маршруты

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка проекта
npm run build
